Hello Web Tech

HTML

HTML is a Language for turning text into hypertext by using markup.

It is the standard markup language for creating web pages.

It is not a programming language - no support for programming constructs.

It is part of the triad of foundational web tech (alongside CSS & javascript).

It describes the semantic structure of the data, which CSS presents, and javascript manipulates.

HTML Elements

HTML documents are constructed from HTML elements.

Elements are keywords encapsulated within angled brackets.
